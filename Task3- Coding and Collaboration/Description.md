# Description: 

In this task I looked into coding practice platforms and collaboration tools to see how they help with academics and skill development.

I started by making an account on HackerRank and doing a beginner problem.

This showed me how these platforms give you practice and help you think logically and solve problems better.

It also taught me about coding challenges and learning based on how you do.

Then I made a Google Form called "Digital Literacy Awareness Quiz" with five questions.

The quiz had choice and short-answer questions to test peoples basic knowledge of digital literacy.

I also checked the answers in Google Sheets, which showed me how data can be collected and analyzed automatically.

These tools are really helpful for students.

HackerRank and similar platforms help you build programming skills.

Google Forms can be used for surveys, quizzes and collecting feedback.

This task gave me hands-on experience with digital tools and showed me how to use them effectively for learning and working together with others.

Coding practice platforms, like HackerRank are useful.

Google Forms and Google Sheets are also tools.
